# Netflix-Movies-vs-TV-Shows-Performance-Analysis-

Project Overview

This project analyzes Netflix’s content dataset to uncover insights about content distribution, growth trends, release patterns, and global expansion. The dashboard provides an interactive and visual representation of Movies vs TV Shows performance, yearly trends, and country-wise distribution.
The goal of this project is to transform raw data into meaningful business insights using data visualization and analytical techniques.


Dashboard Highlights

📽 Movies vs TV Shows Distribution
📅 Content Trend by Year
📈 Titles Added Over Time
🌍 Content by Country (Global Map)
📊 Content Mix Evolution Over Years
🎞 Duration Analysis by Type
🗓 Titles & Duration by Release Year


Key Insights

Movies dominate the platform (~70%) compared to TV Shows (~30%).
Content additions peaked during 2018–2019, indicating rapid expansion.
TV Shows have shown steady growth over time.
North America, Europe, and parts of Asia contribute the highest content volume.
Recent years show a more balanced content mix strategy.
